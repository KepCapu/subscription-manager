# PRD

## Product
Subscription Manager

## Product goal
A mobile app that helps users see all subscription-related spending across linked cards in one place.

## Core problem
Users do not clearly understand:
- which subscriptions are active
- how much they spend per month
- which card is being charged
- how subscription spending changes over time
- whether there are suspicious recurring payments

## Core value
The user opens one app and immediately sees:
- total monthly subscription spending
- active subscriptions
- spending by card
- top expensive subscriptions
- yearly spending trend
- subscription spending distribution
- suspicious recurring payments

## Target user
A person with multiple bank cards and multiple digital subscriptions who wants a simple overview of recurring spending.

## Main screens
- Overview
- Cards
- Card Details
- Subscriptions
- Profile

## MVP outcome
The app should allow the user to understand:
1. how much total money is spent on subscriptions
2. which subscriptions cost the most
3. which cards are used for subscription charges
4. how subscription spending changes over time
5. whether there are suspicious recurring charges

## Non-goals for MVP
- full banking functionality
- general purchase tracking
- warranties
- invoices and receipts management
- investment analytics
- loans or debt management
