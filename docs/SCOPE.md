# SCOPE

## In scope for MVP
- mobile app
- linked cards view
- total monthly subscription spending
- active subscriptions count
- next 7 days spending estimate
- suspicious recurring payments counter
- top expensive subscriptions block
- yearly spending trend chart
- subscription distribution chart
- list of linked cards with monthly totals
- card details screen
- subscription list screen
- profile/settings screen
- mock or initial recurring subscription logic
- clean and consistent UI

## Out of scope for MVP
- full open banking production rollout
- email parsing
- receipt scanning
- cancellation flows
- refund flows
- push notification automation
- family accounts
- admin panel
- advanced fraud detection
- multi-currency accounting beyond basic display
- desktop/web app

## MVP principle
Build the simplest useful product that answers:
- how much do I spend on subscriptions?
- which subscriptions cost the most?
- which cards are charged?
